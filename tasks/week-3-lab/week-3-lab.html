Comparing Both Formats and Rendering the MEI File

A difference between how these file formats present data would be note duration, for example:


MusicXML: note duration is an element which displays the intended duration, e.g. <duration>...</duration>

MEI: note duration is an attribute of an element, e.g. <note dur="..."/>


Some other differences between both filetypes would include elements such as pitch, type, time signature, etc.
